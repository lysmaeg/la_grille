# LA GRILLE

## INSTALLATION

### LINUX

Paquets nécéssaires : 
cmake - apt install cmake

Deux solutions possibles : 

1- Exécuter le fichier install.sh

2- Exécuter les commandes suivantes une par une :

mkdir build
cd build
cmake ..
make


### Windows


1- Exécuter le fichier install.bat

2- Exécuter les commandes une par une :

mkdir build
cd build
cmake ..
make
